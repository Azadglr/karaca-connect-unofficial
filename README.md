# Karaca Connect Unofficial

Unofficial Home Assistant integration for Karaca Connect devices.

Developed by AzadGLR.

## Features

- Tea brewing
- Boiling water
- Filter coffee
- Baby water
- Toggle switch controls
- Turkish status sensor
- Notification and voice settings
- Home Assistant UI setup flow

## Supported Devices

Tested with:

- Karaca Çaysever Robotea Pro Connect 4in1
- Device type: `robotea4in1`

## Installation

This is a private paid integration.

Manual installation:

1. Copy `custom_components/karaca_connect` into `/config/custom_components/`.
2. Restart Home Assistant.
3. Go to Settings → Devices & Services → Add Integration.
4. Search for `Karaca Connect Unofficial`.
5. Enter your Karaca Connect account information.
6. Select your device if multiple devices are found.
